Website Policy

Welcome to our platform! This policy outlines important information about our website, services, and data handling practices.

About Us

Our website serves as a centralized platform for various services deployed on different platforms like AWS, Vercel, and GitHub Pages. Users can access these services conveniently in one place after logging in. The website utilizes Google Firebase for user authentication and Google Analytics for performance tracking.

Services Offered

1. Old Car Price Prediction:

        Get insights about the estimated value of your old cars.

2. Weather Info:

        OpenWeatherMap Api is used to fetch current weather data and displays it on Vibrant web UI, having geolocation button feature.

3. TheNewsBlocks:

        User can Check the latest news fetched from gNews Api on a modern web UI.

4. Currency Convertor:

        Allow users to convert currencies based on real-time exchange rates.

User Authentication

Users are required to log in to access and activate the services. Buttons related to services remain inactive until a user logs in, ensuring privacy and security.

Data Handling

1. User Data

        We collect and store user data securely through Google Firestore for authentication purposes.

2. Analytics

        We use Google Analytics to gather performance metrics and improve our services. Your data is anonymized and used for statistical analysis only.

Responsiveness

Our website is fully responsive, providing a seamless experience across various devices such as mobile phones, tablets, and desktops.
